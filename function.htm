首頁模組function

Function

檔案名稱：sass/_function.scss

範例

                    
<div class="function">
    <button type="button" class="back">回上一頁</button>
    <button type="button" class="print">友善列印</button>
    <button type="button" class="forward">轉寄友人</button>
</div>

範例 - 加入icon

                    
<div class="function">
    <button type="button" class="back icon_back">回上一頁</button>
    <button type="button" class="print icon_print">友善列印</button>
    <button type="button" class="forward icon_forward">轉寄友人</button>
</div>
