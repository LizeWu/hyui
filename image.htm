Images

檔案名稱：sass/_image.scss

圖片影片外框為16:9 或是 4:3，圖片皆能呈現object-fit

【多圖】相簿列表排列與圖片比例

大部分情況下，相簿列表封面套用object-fit的設定以符合圖檔滿版的視覺效果，因此提供此相簿封面快速的SASS以及Jquery設定，html結構沒有固定結構，只須注意imgOuter與img的相對結構即可。

imgOuter為目前設定之classname，如需更動，請注意JS的selector也要相對應。

class="img_circle"
class="img_rounded" 
                        
<img src="..." alt="">
<img src="..." alt="" class="img_circle">
<img src="..." alt="" class="img_rounded">//可用變數控制導角數值
                        
<div class="yourClassName">
    <div class="imgOuterDiv"><img src="..." alt="..."></div>
    <div class="imgOuterDiv"><img src="..." alt="..."></div>
    <div class="imgOuterDiv"><img src="..." alt="..."></div>
</div>
                            
.yourClassName{
    @include make_img_outer();//設定overflow-hidden，非必要
    .imgOuterDiv{
        //@include make_img_col(電腦版欄數,手機版欄數); 
        @include make_img_col(3,2);    //pc 4欄，mobile 2欄
        //@include aspect-ratio(長, 寬);
        @include aspect-ratio(1, 1);   //設定比例ex:4:3 or 16:9
    }
}

JQuery設定

全網站的imgOuter底下的img皆可以使用object-fit Jquery設定

                            
jQuery(document).ready(function() {
    $(window).on('resize load', function(e) {
        $('.imgOuter').each(function(index, el) {
            var _imgContainer = $(this),
            cWidth = _imgContainer.width(),
            cHeight = _imgContainer.height(),
            ratioC = cWidth / cHeight,
            _img = _imgContainer.find('img');

            var iWidth = $(this).find('img').width(),
            iHeight = $(this).find('img').height(),
            ratioImg = iWidth / iHeight,
            scaleRatio;
            if (ratioC > ratioImg) {
                scaleRatio = cWidth / iWidth;
                _img.width(cWidth).height(iHeight * scaleRatio).css('top', -.5 * (iHeight * scaleRatio - cHeight));
            } else {
                scaleRatio = cHeight / iHeight;
                _img.height(cHeight).width(iWidth * scaleRatio).css('left', -.5 * (iWidth * scaleRatio - cWidth));
            }
             $(this).find('img').removeClass('img-responsive');
        });
    });
});
